# codemodel.md SPEC v1.0

File MUST list ALL functions/classes/structures in current folder.

Structure:

# CodeModel – `<name>`

Path: `<relative/path>`

## 1.Elements

For each element (function/class/struct):

- Name
- Type (function|class|struct|interface)
- Signature
- Description (1 line)
- Location `<relative/path/file>`

## 2.Relations

Optional minimal bullets describing:

- Dependencies (calls/uses)
- Exposed-to (who imports/uses it)

## 3.Changelog

Lines: `YYYY-MM-DD – agent – change`
