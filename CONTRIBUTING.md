# Contributing to ASDP

Thank you for your interest in contributing to the Agentic Spec Driven Programming (ASDP) specification!

## Getting Started

1. Read the `README.md` to understand the core concepts.
2. Review the specification files in `spec/asdp/`.

## Contributing Changes

1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Make your changes.
4. Submit a Pull Request.

## Code of Conduct

Please be respectful and constructive in all interactions.
